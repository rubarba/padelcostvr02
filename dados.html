Recolha de Dados

Esta página explica de forma clara e transparente que dados recolhemos, para que servem e como os protegemos.

Que dados recolhemos?

Dados de navegação
Quando visita o PadelCost, recolhemos automaticamente dados sobre a sua visita através do Google Analytics 4: páginas visitadas, duração da sessão, dispositivo utilizado, país de origem e origem do tráfego (pesquisa, directo, redes sociais, etc.).

Endereço de email
Apenas se subscrever voluntariamente a nossa newsletter. Estes dados são utilizados exclusivamente para envio de comunicações relacionadas com o PadelCost.

Dados técnicos
Endereço IP, tipo de browser, sistema operativo e resolução de ecrã — recolhidos automaticamente para fins de análise e segurança.

Para que utilizamos estes dados?
— Melhorar a experiência de navegação e a relevância do conteúdo;
— Compreender quais os produtos e categorias mais consultados;
— Optimizar o desempenho técnico do site;
— Enviar newsletters (apenas com consentimento).

Quanto tempo guardamos os dados?
Os dados analíticos são guardados durante 14 meses no Google Analytics, de acordo com as suas políticas padrão. Os endereços de email da newsletter são guardados até cancelar a subscrição.

Não fazemos
— Não vendemos os seus dados;
— Não partilhamos dados pessoais identificáveis com terceiros;
— Não utilizamos os seus dados para publicidade direccionada fora do nosso site.

Contacto
Para qualquer questão relacionada com os seus dados, contacte: info@padelcost.pt
